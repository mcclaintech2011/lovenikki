Ghost Fire Event

  • This event is pretty straight forward. Collect between 1 - 3 ghost fire per main level that you do. The more ghost fire you can get, the more items you will be able to earn. To earn both the Tengu and Zashiki sets you will need 3800 ghost fire. which is anywhere between 3500 - 1300 stamina. Since the even period is stort this will likely require the use of diamonds.



Haunted Night Time
Limited Draw

  • This is to earn the Shutendoji set; you will have to draw pieces from a gatcha. The first draw is 50 diamonds, subsequent draws are 100 diamonds. Without any duplicates you would need a total of 1650 diamonds to complete the set.






Banshee Momiji
Raffle

  • This is for the Banshee Momiji set and only goes to the top ranking of players. Ranking is determined by how many ghost fire have been earned. The prize distribution is as follows.
  • Top 3000 - Set w/ normal pose
  • Top 1500 - Set w/ special pose
  • Top 500 - Set w/ special pose + background
